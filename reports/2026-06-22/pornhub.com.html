Context

Pornhub is the most-trafficked adult-video platform, a YouTube-style tube mixing user uploads with studio and verified-creator content. It is owned by Aylo — the company that rebranded from MindGeek in August 2023 after Ottawa private-equity firm Ethical Capital Partners acquired it on March 16 2023. Money comes from three layers: free streaming monetized through the in-house TrafficJunky ad network, the Pornhub Premium subscription, and Modelhub, a creator marketplace that pays uploaders and sells pay-per-view clips. For an index builder the defining fact is the December 2020 purge: after a New York Times exposé and the loss of Visa/Mastercard processing, Pornhub deleted every video from unverified uploaders, cutting its library from 13.5M to 2.9M (Verified Metrics). The corpus is now verified-uploader-only, and — unusually — a free public webmasters API serves structured per-video metadata without auth (Probe Log). The second defining fact is legal: age-verification laws now wall the site off in 23 US states, a posture the Supreme Court blessed in June 2025.

Retrieval Notes (site-specific deltas)

Only where Pornhub differs from the default platform assumptions.

Identifier & dedup

The viewkey (returned as video_id) is the stable key, printed in every /view_video.php?viewkey=… URL and joining to thumbs, tags, stars and categories in one API record. Dedup against facet/param aliases — robots.txt blocks dozens of sort/filter variants precisely because they duplicate the same video set.

Retrieval unit

The video is the unit, and unusually the public API hands you a near-complete index document per unit (title, duration, views, rating, tags, stars, categories) in a single call — no page render needed. The expensive parts (the stream, comment text) are exactly what the API omits.

Corpus shape

inferred The pre-2021 long tail is largely gone, so this is a young, curated corpus rather than a 2007-deep archive. Basis: the Dec 2020 purge deleted all unverified uploads (13.5M→2.9M, Verified Metrics); only verified creators can upload since.

Authoritative content field

Titles are SEO-stuffed for adult-search ranking and are the least trustworthy field. The curated categories (241-term taxonomy) and tags are closer to ground truth; rating/views are the quality signal but an adversarial one.

Freshness & recrawl

Deletion is bimodal: trickle removals plus rare mass purges driven by legal/payment events (2020). Recrawl for catalog churn daily (sitemap lastmod), but expect step-function losses an incremental crawler will read as a corpus collapse.

Geographic fragmentation

inferred What the index sees depends on crawl origin: 23 US states get a block page, not the catalog. Basis: Aylo blocks rather than age-checks in those states (Verified Metrics). A US-IP crawl and a Canadian/EU crawl return different surfaces.

Internal vs external search

External engines get a daily curated sitemap and may index view_video pages, but the RTA label, age-gating, and state blocks degrade external retrieval. The on-site search + 241-category taxonomy is the real index; the webmasters API is its public mirror.

User query profile

Navigational and attribute-faceted: category, pornstar name, and body/act descriptors. Long-tail discovery rides the taxonomy, not free-text relevance — which is why the API exposes tags/categories as first-class arrays.

Indexing Plan

  • Retrieval unit: the video, keyed by viewkey; one API record ≈ one ready-made index document.
  • Primary keys & joins: viewkey joins title/tags/thumbs/rating; pornstars[] and categories[] (with their own id endpoints) are the entity joins; channel id links studio/creator.
  • Ingestion paths, ranked: 1) Webmasters search API — full metadata, no auth, but undocumented rate limit and no comment/stream payload. 2) Sitemaps — exhaustive viewkey enumeration (~4.1M) to seed the API sweep. 3) Official embed — the only sanctioned way to surface the actual video. 4) Stream/comment text — no legitimate bulk path; design without them.
  • Recrawl cadence: sitemap diff daily for adds/drops; re-pull API records for high-view videos weekly (views/rating churn); taxonomy + star roster monthly.
  • Derivatives to generate: tag/category normalization, thumbnail (not stream) perceptual hashes for near-dup detection, entity table for the 23,250 stars + 241 categories.
  • Demo to build: a webmasters-API harvester — page search, emit JSONL index docs, and chart how fast you hit a rate limit (the one undocumented constraint) and which fields force you back to page rendering.

Risks to the Index

  • Geographic walls are spreading, not receding: 23 states blocked (Verified Metrics) and the Supreme Court upheld age-verification in FSC v. Paxton (6–3, Jun 27 2025), so more blocks are likely. A single-origin crawl will silently mis-measure availability.
  • Corpus can drop by millions overnight: the 2020 purge erased 78% of the library in days. Legal or payment-processor pressure, not gradual decay, is the dominant deletion driver — version every snapshot.
  • inferred The public API is a courtesy that can be revoked. Basis: the webmasters search/stars/categories endpoints served full metadata with no auth on 2026-06-22; nothing contractual guarantees it stays open or unmetered.
  • inferred AI-crawler access is selectively open today and trivially closable. Basis: robots.txt (2026-06-22) blocks Bytespider/CCBot/Cohere-ai/Anthropic-ai by name but omits GPTBot, ClaudeBot, Google-Extended and PerplexityBot — an allowance that is one edit from reversal.
  • Media URLs expire: thumbnail and stream URLs carry hdnea HMAC tokens (Probe Log), so no durable hotlink — store the viewkey and re-resolve, never the CDN URL.
